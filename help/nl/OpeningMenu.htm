Hoofdmenu

Knoppen:

  • GPX Bestanden

    Met behulp van deze knop worden de invoerbestand(en) (gpx) geselecteerd. Meerdere bestanden kunnen worden geselecteerd door de ctrl-toets ingedrukt te houden.

  • Uitvoerdirectory

    Verwijst naar de directory waar de gegenereerde samenvattingsbestand (.csv) wordt bewaard. A proposal for the "Output filename" is made, which can be changed.

    • Add to file

      The summary of a choosen gpx is added to the existing summary csv file.

    • Summarise
      For the selected GPX-File(s) a summary is generated and written to the summary csv file.
