Opening menu

Buttons:

  • GPX File(s)

    With this button, the input file(s) (gpx) is/are selected. Multiple files can be selected by holding down the ctrl key.

  • Output folder

    Points to the directory where the generated Summary csv-file is stored. A proposal for the "Output filename" is made, which can be changed.

    • Add to file

      The summary of a choosen gpx is added to the existing summary csv file.

    • Summarise
      For the selected GPX-File(s) a summary is generated and written to the summary csv file.
